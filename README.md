Objetivo General

Diseñar e implementar un sistema de inventario de escritorio que permita administrar productos de forma eficiente, controlando existencias, precios y movimientos de entrada y salida.

Objetivos Específicos

- Registrar productos nuevos en el sistema.
- Consultar información de productos almacenados.
- Modificar datos de productos existentes.
- Eliminar productos del inventario.
- Controlar cantidad disponible de cada producto.
- Generar reportes básicos de inventario.

Justificación

Actualmente muchos pequeños negocios realizan el control de inventario de forma manual, lo que puede generar errores en cantidades, pérdidas de información o desorganización.
Con este sistema se busca ofrecer una solución sencilla que permita:
- Mejor organización
- Rapidez en consultas
- Reducción de errores
- Mejor control de existencias

Entorno de Trabajo a Utilizar

- Lenguaje: Java
- IDE: Apache NetBeans
- Base de datos: SQLite
- Control de versiones: GitHub
